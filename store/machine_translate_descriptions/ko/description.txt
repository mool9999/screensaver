이 확장 프로그램은 다양한 소스에 있는 사진을 슬라이드 쇼로 구성하여 스크린 세이버로 표시합니다. 지원되는 사진 소스는 다음과 같습니다.

* 당신의 구글 포토와 앨범들 - 참고 : 공유된 앨범은 지원되지 않습니다.
* Chromecast에서 사용한 배경 사진 일부
* Reddit에 있는 다양한 소스
* Flickr의 흥미로운 사진

참고 : 로컬 파일 시스템에 있는 사진은 지원되지 않습니다.

슬라이드 쇼를 제어하기위해 사용할 수 있는 옵션들은 다음과 같습니다.

* 화면 보호기를 표시하기 위한 대기 시간 조정
* 사진과 사진의 시간 간격
* 사진 표시 모드 (예 : 레터 박스, 화면 채우기 등)
* 사진 전환 애니메이션 - (예 : 페이드, 스케일 업)
* 스크린 세이버, 디스플레이 및 컴퓨터의 동작 스케쥴 제어
* Chrome 창이 전체화면일 때(예 : YouTube, Netflix) 표시유무
* 사용자의 모든 디스플레이에 표시
* 현재 날씨 표시
* 현재 시간 표시

모든 플랫폼에서 훌륭하게 작동하지만 Chromebook에 특히 유용 할 것입니다.

이 로그램은 무료이며 광고도 없습니다.
